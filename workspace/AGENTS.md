# 에이전트 지침

## 지도 헬퍼

다음 조건일 때, 지도 링크 생성 처리를 진행.

- 입력 메시지가 `[지도 링크 감지]`로 시작.
- 사용자가 직접 링크 생성을 요청. (예: 방금 보낸 주소 네이버 링크로 만들어 줘)

**내비용 주소**는 시/도를 뺀 **시군구 이하 도로명 주소만** 사용한다.

예시:
- 입력: `서울 강남구 선릉로 605 HJ빌딩 2층` → 출력: `강남구 선릉로 605`
- 입력: `경기 성남시 분당구 판교역로 235` → 출력: `분당구 판교역로 235`

### 지도 링크 응답 형태

지원하는 지도 응답 형식은 4가지. 응답은 반드시 아래 템플릿 형식만 사용한다. URL이나 주소를 직접 작성하지 않는다.

1. `{{주소:내비용 주소}}` → 내비용 주소 텍스트 (복붙용)
2. `{{카카오지도:내비용 주소}}` → 카카오맵 링크
3. `{{네이버지도:내비용 주소}}` → 네이버지도 링크
4. `{{티맵:내비용 주소}}` → 티맵 링크

### 지도 링크 생성 절차

#### 메시지에 링크 없이 주소만 있을 때

내비용 주소 생성 후, 카카오, 네이버, 티맵 응답을 생성하여 반환.

#### 메시지에 특정 서비스 지도 링크가 존재할 때

내비용 주소 생성 후, 내비용 주소와 그 외 서비스의 지도 링크드를 생성

예: 사용자가 네이버 링크를 보내면, 내비용 주소 텍스트 및 타 서비스 (카카오, 티맵) 링크를 전달.
